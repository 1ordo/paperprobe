# Reference Documents

This directory contains reference materials for the COSMIN methodology.

## Required Document

Download the COSMIN Risk of Bias user manual (v4) from the official source:
- URL: https://www.cosmin.nl/wp-content/uploads/user-manual-COSMIN-Risk-of-Bias-tool_v4_JAN_final.pdf
- Save as: `COSMIN_RoB_Manual_v4.pdf` in this directory

This PDF is not included in the repository due to copyright. The AI agents embed
the manual's rating rules directly in their prompts, so the PDF is only needed for
human reference.
